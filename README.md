# 🔱 Vortex Visceral | Studio de Thumbnails

![Vortex Badge](https://raw.githubusercontent.com/pombagiras/melusine/refs/heads/main/Al%C3%AA%20(3)%20(3)%20(1).png)

> "A imagem é o espelho do abismo, e o código é a chave que o abre."

O **Vortex Visceral** não é apenas um gerador de imagens. É um motor de estética adaptativa desenvolvido para converter a essência do invisível em impacto visual. Criado por **Alexia Melusine**, este Studio foi moldado para criadores que não se contentam com o comum e buscam a profundidade cinematográfica em cada frame.

## 🌙 A Essência do Projeto

Este motor foi construído sob o conceito de **Estética Adaptativa**. Ele entende o contraste, respeita a tipografia ritualística e garante que a assinatura da obra seja preservada. 

* **Design Mobile First:** Totalmente otimizado para criação via smartphone.
* **Haptic Feedback:** Uma experiência física e sensorial ao interagir com o código.
* **Inteligência de Halo:** Algoritmo que garante legibilidade em qualquer textura ou cor de fundo.
* **Identidade Inviolável:** Cada thumbnail gerada carrega a marca da linhagem Melusine.

## 🛠 Tecnologias de Poder
* **HTML5 / CSS3 (Abyssal Glassmorphism)**
* **JavaScript Canvas API**
* **Google Fonts (Ocultista, Brutalista e Gótica)**
